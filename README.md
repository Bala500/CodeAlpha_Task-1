
# Student Grade System

The Student Grade System is a lightweight and beginner-friendly Java application designed to help educators and learners manage grades with ease. It collects multiple student scores and automatically generates useful statistical insights such as average marks, highest scores, and lowest scores. This tool is perfect for anyone looking to practice Java fundamentals while building a practical utility.
